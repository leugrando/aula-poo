## Comandos importantes para lembrar

### Clonar repositorio
git clone {endereço}

### Para adicionar arquivos no stage
git add {nome do arquivo}
ou
git add .

### Para enviar / commitar o codigo
git commit -m "MENSAGEM DO COMMIT"

### Para ver o log
git log

### Para ver a situacao do working dir
git statusx
